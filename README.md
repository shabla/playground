# Formatting with prettier + eslint
- Install VS Code extension `Prettier ESLint` as use it as formatter
- Install devDependencies:
    - `eslint`
    - `eslint-plugin-react`
    - `prettier-eslint`
    - `@typescript-eslint/eslint-plugin`
    - `@typescript-eslint/parser`
- Create `.eslintrc.js`
- Create `prettier.config.js`
